# 🧾 Blockchain-Based Will Executor

A decentralized smart contract system that automates the execution of a will using blockchain technology, ensuring secure, transparent, and tamper-proof asset transfer without intermediaries.

---

## 📌 Overview

This project leverages Ethereum smart contracts to enable automated inheritance execution. The owner can deposit funds into the contract, and upon verification of conditions (e.g., declaration of deceased), assets are securely transferred to beneficiaries.

---

## ⚙️ Features

- 🔐 Secure fund storage using smart contracts
- 👤 Owner-controlled deposit functionality
- 📜 Automated inheritance execution
- ✅ Beneficiary validation and access control
- ⛓️ Tamper-proof and transparent transactions
- 💰 Trustless asset transfer without intermediaries

---

## 🛠️ Tech Stack

- **Blockchain:** Ethereum
- **Smart Contracts:** Solidity
- **Development Environment:** Hardhat
- **Wallet Integration:** MetaMask
- **Frontend:** HTML, CSS, JavaScript (if applicable)

---

## 🧠 How It Works

1. The contract owner deploys the smart contract
2. Funds are deposited into the contract
3. Beneficiaries are predefined
4. Owner declares deceased status
5. Smart contract automatically transfers funds to beneficiaries
